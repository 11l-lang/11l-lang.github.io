Однострочные комментарии



Символы 
//
или 
\\
обозначают начало комментария, который оканчивается символом новой строки.
Символы 
\\
предназначены для обозначения того, что данный комментарий продолжает комментарий из предыдущей строки (пример).


Многострочные и внутристрочные комментарии



Символы 
\(
, 
\[
, 
\{
или 
\‘
обозначают начало комментария, который завершается соответствующим символом 
)
, 
]
, 
}
или 
’
. Эти комментарии могут быть вложенными.
\(
this_function_call_is_commented_out()

// Вложенный комментарий:
\(
// Пожалуйста поддерживайте баланс скобок и других парных символов в вашем исходном коде:
I ch == "(" // )
   ...
) // конец вложенного комментария
)
this_function_call_is_not_commented_out()

[Поддержание баланса парных символов 
()
, 
[]
, 
{}
и 
‘’
рекомендуется не только для корректной работы многострочных комментариев в 11l, но и для корректной подсветки соответствующих скобок в редакторах исходного кода.]

При комментировании большого куска кода можно использовать дополнительные скобки для большей заметности границ комментария:
\[[[
... большой кусок кода ...
]]]
