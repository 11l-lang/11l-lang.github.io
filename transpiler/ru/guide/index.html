Попробуйте исправить данный Python-код так, чтобы он корректно скомпилировался транспайлером Python → 11l → C++:
  1. Mocha и прогулка
  2. Mocha и Diana (простая версия)
  3. Запрещённая подпоследовательность
  4. Задача про НОД
  5. Mocha и красное и синее

Для проверки вашего решения откройте страницу соответствующей задачи (в таблице под словом ‘Задача’), сохраните входные данные примера в файл input.txt и выполните в командной строке следующие 3 команды:
...11l your_solution.py
python your_solution.py < input.txt > output_py.txt
your_solution < input.txt > output.txt
А затем сравните содержимое файлов output_py.txt и output.txt.
